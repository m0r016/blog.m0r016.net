netlifyからgithub pagesに乗り換える

はじめに

netlifyはサイトのロード時間が長いと感じたため、github pagesに乗り換えようと思う。

目次

  • github pagesとは
  • 1.ブランチを作成する
  • 2.hexo-deployer-gitをインストールする

github pagesとは

github pagesとは、githubが運営しているウェブホスティングサービスだ。
netlify同様、githubに上げたリポジトリをもとに、自動的に生成。もちろん送信すると自動的に再生成してくれる。

1.ブランチを作成する

github pagesではリポジトリではなく、ブランチ単位で生成できるため、ブランチを生成する

2.hexo-deployer-gitをインストールする

hexoにデプロイさせるため拡張機能をインストールする

インストールが完了したら、hexo-deployer-gitをもとに設定を行う。_config.ymlを開き、追加していく。

token生成のやり方はここにアクセスし

Generate new tokenをクリック

Select scopes内のrepoをクリック

をクリックするとtokenが生成されるのでコピペするといい。

_config.ymlを保存する。
